管理后台

🤖 AI 模型设置

启用后，人物开会发言和协作讨论将由模型根据画像实时生成（日志中带 ✨ 标记）； 未启用或调用失败时自动使用内置台词。
🔒 API Key 只保存在你这台设备的浏览器里（localStorage），不会上传到任何服务器，调用时由你的浏览器直接请求模型官方接口。

OpenAI 兼容服务的 API 地址，以 /v1 结尾

🏢 公司设定

这里定义「这是个什么公司」。产品有真实运行状态（日活、满意度、Bug、现金跑道），每天演化并被市场事件冲击； 员工知道这些数据和事件，开会时会真的讨论它们。这些设定会进入每个人物的世界认知。

人物的记忆流和产品指标会重置回第 1 天（人物画像和模型配置保留）

🏛️ 上层决策

发布的决策会作为公司公告进入所有人的记忆，并持续约束他们之后的发言，直到撤销。 需要本地 sidecar 运行，并从 sidecar 地址打开本页（如 http://127.0.0.1:7878/admin.html）。

⚠️ 未检测到 sidecar，本区块不可用。启动方式见 README。

🧑‍💼 人物管理

每个人物的画像描述会直接交给模型，写得越具体，人物的发言越有个性。最多 8 人。
