# 画像办公室 · 多 Agent 3D 模拟

一个纯前端的 3D 小办公室模拟：多个由「人物画像」驱动的 3D 小人会自主工作、开会、协作讨论、去咖啡角休息。手机和电脑浏览器都能直接打开，无需任何服务器。

## 功能

- 🏢 **3D 小办公室**：工位区、玻璃会议室（带白板）、咖啡角、休息区
- 🧑‍💼 **画像驱动的 Agent**：每个人物有姓名、职位、性格和专属台词，点击人物可查看画像卡片，镜头会轻轻跟随
- 📅 **每日日程**：09:00 开工 → 10:00 每日站会（轮流发言）→ 12:00 午休 → 15:30 项目评审会 → 17:30 茶歇 → 18:30 下班，循环往复
- 🤝 **随机协作**：工作时间里人物会随机走到同事工位讨论，对话显示在头顶气泡里
- 🗺️ **A\* 寻路**：人物自动绕开桌椅家具行走，不会穿模卡死
- 📜 **办公室动态**：右下角实时滚动的事件日志
- ⏩ **时间控制**：暂停 / 1× / 3× / 8× 加速
- 📱 **手机适配**：触屏拖动旋转、双指缩放，UI 自动适配小屏

## 如何打开

这是纯静态页面（Three.js 走 CDN），任选一种方式：

**方式一：GitHub Pages（推荐，手机随时可访问）**

仓库 Settings → Pages → 选择分支和根目录，保存后访问分配的网址即可。

**方式二：本地预览**

```bash
npx http-server -p 8080
# 然后浏览器打开 http://localhost:8080
```

> 注意：直接双击 index.html（file:// 协议）无法加载 ES 模块，需要用本地服务器。

## 替换成你自己的人物画像

只需编辑 **`js/personas.js`** 一个文件。每个人物的格式：

```js
{
  id: "lin",              // 唯一标识
  name: "林晓",            // 姓名
  role: "产品经理",        // 职位
  color: 0xe05a4e,        // 衣服颜色
  skin: 0xf2c9a4,         // 肤色
  hair: 0x2b1d12,         // 发色
  personality: "……",      // 性格描述（点击人物时展示）
  lines: {
    work:    ["…"],       // 工位上的自言自语
    meeting: ["…"],       // 开会发言
    collab:  ["…"],       // 协作讨论台词
    coffee:  ["…"]        // 咖啡角闲聊
  }
}
```

人数可增减（建议 ≤ 6 人，与工位、会议室座位数一致；超出会复用座位）。每日日程也在同一文件的 `DAILY_SCHEDULE` 中调整。

## 目录结构

```
index.html          页面入口
css/style.css       UI 样式（含手机适配）
js/personas.js      ★ 人物画像与日程配置（要换人改这里）
js/main.js          渲染器、相机、交互、主循环
js/office.js        办公室场景搭建与点位定义
js/agent.js         3D 小人：建模、行走、坐立、气泡、动画
js/director.js      导演：模拟时钟、日程调度、会议与协作事件
js/pathfinding.js   网格 A* 寻路
```
